# Egyptian English Academy — Multi-Level English Learning Platform

> تطبيق ويب تعليمي متكامل لتعلّم الإنجليزية من الصفر للمستويين A1 و A2 باللهجة المصرية

تطبيق صفحة واحدة (SPA) يعمل كاملاً من جانب العميل دون الحاجة لأي خادم، مخصص لتعليم اللغة الإنجليزية للناطقين بالعربية المصرية وفق معايير الإطار الأوروبي المشترك المرجعي للغات (CEFR) للمستويين **A1 Elementary** و **A2 Pre-Intermediate**. يجمع بين واجهة إنجليزية احترافية وشرح تفصيلي بالعامية المصرية داخل كل درس لتبسيط القواعد وربطها بالواقع اليومي.

## نظرة عامة

يهدف التطبيق إلى توفير تجربة تعليمية شاملة ومحفزة تغطي مهارات اللغة الأربع الأساسية:

| المهارة | كيف يُعلّمها التطبيق |
|---|---|
| **الاستماع (Listening)** | جمل صوتية مُركَّبة + بطاقات مطابقة تفاعلية |
| **الكلام (Speaking)** | تحقق صوتي بالميكروفون عبر Web Speech Recognition |
| **القراءة (Reading)** | نصوص قصيرة + أسئلة فهم متعددة الاختيار بشرح عربي مبسط |
| **الكتابة (Writing)** | بناء جمل من كلمات مبعثرة + ملء الفراغات النحوية |

## الهيكل والمنهج الدراسي

تدعم المنصة حالياً مستويين دراسيين كاملين يتم تحميل بياناتهما ديناميكياً بناءً على اختيار المستخدم:

### 1. المستوى A1 (Elementary)
* **عدد الدروس:** 40 درساً موزعة على 10 وحدات (4 دروس لكل وحدة).
* **المفردات:** 10 مفردات لكل درس بإجمالي **1000 مفردة** أساسية.
* **الاختبارات:** 40 اختباراً لتقييم الدروس + اختبار نهائي شامل يتكون من 20 سؤالاً ديناميكياً لتوليد شهادة التخرج.

### 2. المستوى A2 (Pre-Intermediate)
* **عدد الدروس:** 40 درساً موزعة على 10 وحدات (4 دروس لكل وحدة).
* **المفردات:** 30 مفردة لكل درس بإجمالي **1200 مفردة** متقدمة لتغطية النطاق اللغوي الأوسع.
* **الاختبارات:** 40 اختباراً لتقييم الدروس + اختبار نهائي شامل لتوليد شهادة تخرج معتمدة تحمل شعار ورمز المستوى A2.

## المميزات الاحترافية الجديدة

تمت إضافة لوحة إعدادات احترافية متكاملة تتيح للمستخدم إدارة تجربته التعليمية بالكامل:

### 1. زر الإعدادات الاحترافي (Professional Settings)
* تم استبدال القوائم الهامشية التقليدية بزر إعدادات عائم واحترافي يفتح نافذة منبثقة (Modal) منسقة بشكل ممتاز تحتوي على كافة الإعدادات الضرورية.
* يتيح التحكم في: تفعيل/تعطيل المؤثرات الصوتية، تبديل المظهر (Dark/Light Mode)، وضبط سرعة النطق الصوتي للـ TTS.

### 2. تبديل المستوى الدراسي ديناميكياً (Level Switcher)
* إمكانية تغيير المستوى الدراسي النشط (بين A1 و A2 أو أي مستويات تجريبية أخرى) مباشرة من داخل قائمة الإعدادات.
* عند تغيير المستوى، يقوم النظام بتحميل بيانات المنهج الدراسي الجديد ديناميكياً وإعادة تهيئة لوحة التحكم وخريطة الطريق والقاموس فوراً لتتوافق مع المستوى المختار دون تداخل في البيانات.

### 3. النسخ الاحتياطي واستعادة التقدم (Backup & Restore Progress)
* **تصدير التقدم (Export):** يتيح للمستخدم حفظ تقدمه الدراسي ونقاط الخبرة والدروس المكتملة في ملف بصيغة JSON على جهازه الشخصي بضغطة زر واحدة.
* **استيراد التقدم (Import):** يتيح رفع ملف التقدم المحفوظ مسبقاً لاستئناف التعلم من أي جهاز آخر، مع التحقق التلقائي من سلامة هيكل الملف المستورد لمنع تلف البيانات.

## المميزات التفصيلية للمنصة

### 1. لوحة التحكم (Dashboard)
* **حلقة التقدم الدائرية:** رسم ديناميكي دقيق يحسب نسبة إنجاز المستوى الحالي بناءً على عدد الدروس المكتملة الفعلي في هذا المستوى.
* **إحصائيات الأداء:** تتبع نقاط الخبرة (XP) المكتسبة، الدروس المنجزة، والأسئلة المجابة، بالإضافة لرسوم بيانية لمستوى إتقان المهارات الأربع.

### 2. خريطة التعلم التفاعلية (Roadmap)
* **نظام القفل المتتابع:** يُفتح الدرس التالي فقط عند اجتياز الدرس الحالي بنسبة نجاح لا تقل عن 60% في الاختبار وإتمام تلاوة مفرداته.
* **التوافق الديناميكي:** تتكيف الخريطة تلقائياً لعرض عدد الوحدات والدروس الخاصة بالمستوى النشط مع إظهار كرت التخرج عند نهاية آخر وحدة متوفرة للمستوى.

### 3. مركز المفردات والتحقق المزدوج (Vocab Center)
* **النطق:** نطق الكلمة عبر محرك الـ TTS ثم قيام الطالب بنطقها عبر الميكروفون للتحقق الصوتي.
* **التهجئة:** كتابة الكلمة مع فحص دقيق عبر محرك تحليل الأخطاء لتحديد الحروف الناقصة أو الزائدة أو الخاطئة وعرضها للمستخدم.

## المعمارية التقنية وتغيير المستويات

يعتمد التطبيق على بنية برمجية خالية من أطر العمل (Vanilla JS Modules) تعتمد على التوجيه المستند إلى الهاش (Hash-based Routing). 

يتم إدارة المستويات عبر متحكم مستويات مركزي (`js/levelManager.js`) يقوم بالآتي:
1. قراءة المستوى النشط من التخزين المحلي (LocalStorage).
2. استيراد ملف المنهج الخاص بالمستوى المختار ديناميكياً (مثل `js/data/A2/curriculum.js`).
3. تحديث الكائن العالمي `levelData` لجعله متاحاً لكافة المكونات البرمجية لتحديث الواجهات بناءً عليه.

## نظام الـ XP والرتب

يتم احتساب نقاط الخبرة لمنح مكافآت تشجيعية للطالب وتحديد رتبته اللغوية:

| الحدث | نقاط الخبرة المكتسبة (XP) |
|---|---|
| إجابة صحيحة في اختبار الدرس | +20 XP |
| إتمام تلاوة مفردات درس بالكامل | +100 XP |
| إتمام درس كامل (اختبار + تلاوة) | +150 XP Bonus |
| سؤال مراجعة يومية صحيح | +10 XP |
| إتمام بطاقة مفردات منفردة | +5 XP |
| اجتياز الامتحان النهائي للمستوى | +300 XP |
| محاولة الامتحان النهائي (دون اجتياز) | +50 XP |

## نظام التخزين والنسخ الاحتياطي

يتم حفظ التقدم الدراسي محلياً في المتصفح تحت مفتاح موحد لضمان الاستمرارية:
* **مفتاح التخزين الأساسي:** `egyptian_english_academy_progress`
* يتم تخزين مصفوفة الدروس المكتملة، درجات الاختبارات، إجمالي الـ XP، حالة الامتحان النهائي، والاسم الكامل للطالب.
* عند القيام بالنسخ الاحتياطي، يتم سحب هذا الكائن بالكامل وتحويله إلى ملف JSON جاهز للتحميل.

## نظام الصوت والمؤثرات

* **SpeechSynthesis (TTS):** لنطق الجمل والكلمات الإنجليزية بلكنة أمريكية أو بريطانية مع إمكانية تعديل السرعة.
* **SpeechRecognition:** للتعرف على صوت المستخدم وتقييم النطق الإنجليزي.
* **Audio Effects Engine:** محرك صوتي يولد المؤثرات الصوتية (أصوات النجاح، الخطأ، الاحتفال، والنقر) برمجياً عبر الـ Web Audio API دون الحاجة لملفات صوتية خارجية لتقليل وقت تحميل الصفحة.

## التقنيات المستخدمة

* **HTML5:** لبناء الهيكل وتوفير واجهات متوافقة مع متطلبات الوصول.
* **CSS3 (Vanilla):** للتصميم العصري ثلاثي الأبعاد ودعم وضعي القراءة (الداكن والفاتح).
* **JavaScript (ES Modules):** لتقسيم الكود البرمجي وإدارته بكفاءة دون مكتبات خارجية.
* **Web APIs:** (SpeechSynthesis, SpeechRecognition, Web Audio, LocalStorage).

## تشغيل المشروع محلياً

المشروع عبارة عن تطبيق ويب ثابت يعمل بالكامل في المتصفح. لتجربة كافة الميزات بما فيها الميكروفون، يوصى بتشغيله عبر خادم محلي:

**باستخدام Node.js:**
```bash
npx serve .
```

**باستخدام Python:**
```bash
python -m http.server 3000
```
ثم افتح الرابط `http://localhost:3000` في متصفح حديث مثل Google Chrome أو Microsoft Edge.
